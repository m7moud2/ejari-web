## ⚠️ المشاكل المعروفة والحلول المقترحة

### 1. صفحات الخدمات (أولوية متوسطة)
**الصفحات:** `cleaning.html`, `maintenance.html`, `moving.html`, `insurance.html`, `rent-guarantee.html`

**المشكلة:** أزرار الحجز قد لا تحفظ البيانات بشكل صحيح.

**الحل المقترح:**
```javascript
function bookService(serviceName, price) {
    const booking = {
        id: Date.now(),
        itemTitle: serviceName,
        price: price,
        totalCost: price,
        type: 'service',
        startDate: new Date().toISOString(),
        image: 'images/service-icon.png'
    };
    localStorage.setItem('currentBooking', JSON.stringify(booking));
    window.location.href = 'payment.html';
}
```
