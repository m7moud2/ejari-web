# دليل استخدام لوحة تحكم المالك (Owner Dashboard)

تم تحديث لوحة التحكم لتشمل ميزات احترافية تساعدك في إدارة عقاراتك وإيراداتك بكفاءة.

## الميزات الجديدة:

### 1. نظرة عامة شاملة (Overview)
- **بطاقات الإحصائيات:** تعرض عدد العقارات، الحجوزات النشطة، الإيرادات الشهرية، والطلبات المعلقة.
- **رسم بياني للإيرادات (Revenue Chart):** يعرض أداء الإيرادات خلال آخر 6 أشهر بشكل تفاعلي.

### 2. إدارة الطلبات (Request Management)
- يمكنك رؤية طلبات الحجز المعلقة من المستأجرين.
- إمكانية **قبول** أو **رفض** الطلبات بضغطة زر.
- عند القبول، يتم تحديث الحالة وإشعار المستأجر للدفع.

### 3. إدارة العقارات (Property Management)
- إضافة عقارات جديدة بسهولة.
- تعديل بيانات العقارات الحالية (قريباً).
- عرض حالة كل عقار (مؤجر، متاح، صيانة).

### 4. المحفظة (Wallet)
- تتبع جميع المعاملات المالية.
- طلب سحب الأرباح.
