# 📋 دليل منصة إيجاري الشامل

## 🎯 نظرة عامة
**إيجاري** هي منصة متكاملة لإدارة الإيجارات في الوطن العربي، تجمع بين تأجير العقارات والسيارات وتقديم الخدمات المساندة.

---

## 📄 الصفحات الرئيسية

### 🏠 **الصفحة الرئيسية** (`index.html`)
- عرض الخدمات الرئيسية
- إحصائيات المنصة
- أحدث العقارات والسيارات
- نظام الترجمة (عربي/إنجليزي)
- تحويل العملات (SAR, EGP, AED, USD)

### 🔐 **صفحات المصادقة**
- `login.html` - تسجيل الدخول
- `signup.html` - إنشاء حساب جديد
- `forgot-password.html` - استعادة كلمة المرور

---

## 🏢 **صفحات العقارات**

### للمستأجرين:
- `properties.html` - تصفح جميع العقارات المتاحة
- `property-details.html` - تفاصيل العقار مع زر الحجز
- `tenant-dashboard.html` - لوحة تحكم المستأجر
- `tenant-profile.html` - الملف الشخصي للمستأجر

### للملاك:
- `owner-dashboard.html` - لوحة تحكم المالك
- `owner-profile.html` - الملف الشخصي للمالك
- `add-property.html` - إضافة عقار جديد
- `details.html` - إدارة تفاصيل العقار

---

## 🚗 **صفحات السيارات**

- `cars.html` - تصفح السيارات المتاحة للإيجار
- `car-details.html` - تفاصيل السيارة مع زر الحجز

## 🎨 **الميزات التقنية**

### 🌍 **نظام التوطين** (`js/localization.js`)
- تبديل اللغة (عربي ⟷ إنجليزي)
- تحويل العملات التلقائي
- تغيير اتجاه النص (RTL/LTR)
- دعم 4 دول: السعودية، مصر، الإمارات، الولايات المتحدة

### 🔄 **الترجمة التلقائية** (`js/auto-translator.js`)
- قاموس شامل يحتوي على +400 مصطلح
- ترجمة تلقائية لجميع عناصر الواجهة
- دعم النصوص، Placeholders، والسمات

### 💾 **إدارة البيانات** (`js/dashboard.js`)
- تخزين محلي (LocalStorage)
- إدارة الحجوزات والعقارات
- نظام النقاط والمكافآت
- سجل المعاملات المالية

### 🎁 **نظام المكافآت**
- **للمستأجرين:**
  - 100 نقطة لكل حجز
  - 50 نقطة لكل تقييم
  - 200 نقطة لدعوة صديق
  - استبدال النقاط بخصومات أو قسائم

- **للملاك:**
  - 100 نقطة لكل حجز جديد
  - استبدال النقاط بأموال في المحفظة
  - خصومات على رسوم الاشتراك

## 🔗 **مسارات المستخدم الرئيسية**

### 1️⃣ **حجز عقار (مستأجر)**
```
properties.html 
  → property-details.html 
    → payment.html 
      → success.html 
        → tenant-dashboard.html
```

### 2️⃣ **حجز سيارة (مستأجر)**
```
cars.html 
  → car-details.html 
    → payment.html 
      → success.html 
        → tenant-dashboard.html
```

### 3️⃣ **إضافة عقار (مالك)**
```
owner-dashboard.html 
  → add-property.html 
    → owner-dashboard.html (يظهر العقار الجديد)
```

### 4️⃣ **الاشتراك في باقة (مالك)**
```
subscriptions.html 
  → payment.html?plan=pro&type=owner 
    → success.html 
      → owner-dashboard.html
```
