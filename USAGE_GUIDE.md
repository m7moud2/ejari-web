# 📖 دليل استخدام منصة إيجاري

## 🎯 نظرة عامة

هذا الدليل يشرح كيفية استخدام موقع **إيجاري** لعرض فكرة البيزنس على المستثمرين والعملاء المحتملين.

---

## 📂 ملفات المشروع الرئيسية

### 1. **index.html** - الصفحة الرئيسية المحسّنة
الصفحة الرئيسية تحتوي على:
- قسم Hero محسّن بتصميم جذاب
- قسم الإحصائيات (Stats) مع أرقام مثيرة
- قسم "عن المنصة" (About)
- قسم المميزات (Features) مع 6 مميزات رئيسية
- قسم "كيف تعمل المنصة" (How It Works)
- قسم دعوة للعمل (CTA) للمستثمرين
- قسم العقارات والسيارات المتاحة

**كيفية الاستخدام:**
```bash
# افتح الملف في المتصفح
open index.html
# أو
file:///Users/mahmoudabdelkawy/ejari-web1/index.html
```

### 2. **investor-pitch.html** - العرض التقديمي للمستثمرين
عرض تقديمي احترافي بشاشة كاملة يحتوي على 8 شرائح:

1. **الشريحة 1**: الغلاف - اسم المنصة والشعار
2. **الشريحة 2**: المشكلة - التحديات في سوق الإيجارات
3. **الشريحة 3**: الحل - كيف تحل إيجاري المشكلة
4. **الشريحة 4**: حجم السوق - الأرقام والإحصائيات
5. **الشريحة 5**: نموذج العمل - مصادر الدخل
6. **الشريحة 6**: التوقعات المالية - جدول الأرقام
7. **الشريحة 7**: فرصة الاستثمار - التمويل المطلوب
8. **الشريحة 8**: دعوة للعمل - معلومات التواصل

**كيفية التنقل:**
- استخدم الأسهم (↑ ↓ ← →) للتنقل بين الشرائح
- انقر على النقاط على الجانب الأيسر للانتقال لشريحة معينة
- قم بالتمرير بالماوس للتنقل

**نصائح للعرض:**
- اضغط F11 لوضع الشاشة الكاملة
- استخدم جهاز عرض (Projector) للعروض الكبيرة
- تدرب على العرض قبل الاجتماع

### 3. **BUSINESS_PITCH.md** - وثيقة العرض التقديمي
ملف Markdown شامل يحتوي على:
- نظرة عامة على المشروع
- المشكلة والحل
- حجم السوق
- نموذج العمل
- خطة النمو (3 سنوات)
- التمويل المطلوب
- التوقعات المالية
- معلومات الفريق
- استراتيجية الخروج

**كيفية الاستخدام:**
- اطبع الملف وقدمه للمستثمرين
- شاركه عبر البريد الإلكتروني
- استخدمه كمرجع أثناء الاجتماعات

### 4. **README.md** - دليل المشروع
ملف توثيق شامل للمشروع يحتوي على:
- نظرة عامة
- المميزات
- التقنيات المستخدمة
- هيكل المشروع
- خارطة الطريق
- معلومات التواصل

```css
/* ألوان الخلفيات */

```

- **الأوزان المستخدمة**: 400, 500, 600, 700, 800

## 🚀 السيناريوهات المختلفة للاستخدام

### السيناريو 1: اجتماع مع مستثمر
1. افتح `investor-pitch.html` في وضع الشاشة الكاملة
2. ابدأ بالشريحة الأولى (الغلاف)
3. انتقل عبر الشرائح بالترتيب

1. افتح `index.html`
2. ابدأ بقسم Hero
3. اشرح المميزات في قسم Features
4. أرِ كيف تعمل المنصة في قسم How It Works
5. اعرض العقارات المتاحة

**المدة المقترحة**: 10-15 دقيقة

### السيناريو 3: إرسال معلومات عبر البريد
1. أرسل رابط الموقع: `index.html`
2. أرفق ملف `BUSINESS_PITCH.md` كـ PDF
3. أضف ملخص تنفيذي في البريد
4. حدد موعد لاجتماع متابعة
