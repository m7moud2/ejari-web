منصة إيجاري — تطبيق الجوال

الإصدار الحالي للتطبيق 1.3.3+

باستخدامك لتطبيق إيجاري فإنك توافق على جمع واستخدام المعلومات كما هو موضح أدناه. هذه السياسة مطلوبة لنشر التطبيق على Google Play.

1. البيانات التي نجمعها

  • بيانات الحساب: الاسم، البريد الإلكتروني، رقم الهاتف، نوع الحساب (مستأجر / مالك / فني).
  • بيانات العقارات والحجوزات والمعاينات والصيانة والمدفوعات داخل التطبيق.
  • الموقع التقريبي (بإذنك) لعرض أقرب الوحدات المتاحة.
  • صور اختيارية للتوثيق أو بلاغات الصيانة (بالكاميرا).
  • بيانات تقنية أساسية عن الجهاز لتشغيل الإشعارات وتحسين الاستقرار.

2. كيف نستخدم البيانات

  • إنشاء الحساب وإدارة الحجوزات والعقود والصيانة.
  • معالجة المدفوعات عبر مزودي دفع معتمدين (مثل Paymob) عند تفعيلها.
  • إرسال إشعارات متعلقة بالحجز أو الدفع أو الدعم.
  • تحسين الأمان ومنع الاحتيال وإساءة الاستخدام.

3. المدفوعات

مدفوعات الإيجار/العربون/الخدمات العقارية تُعالَج كخدمات مرتبطة بعقار حقيقي. عند استخدام بوابة دفع خارجية، تُشارك بيانات الفوترة اللازمة مع مزود الدفع وفق سياساته الأمنية. لا نخزّن أرقام البطاقات كاملة على خوادم إيجاري.

4. المشاركة مع أطراف ثالثة

  • Firebase (Google) للمصادقة وتخزين البيانات عند تفعيل الوضع الحقيقي.
  • مزودو الدفع المعتمدون لإتمام المعاملات.
  • خرائط Google عند استخدام البحث بالموقع/الخريطة.

لا نبيع بياناتك الشخصية لأطراف لأغراض تسويقية.

5. الأمان وحقوقك

نتخذ إجراءات تقنية وتنظيمية معقولة لحماية بياناتك. يمكنك طلب تصحيح أو حذف بياناتك عبر الدعم داخل التطبيق أو عبر البريد الظاهر في صفحة التواصل.

6. أذونات التطبيق

  • الإنترنت: للتشغيل والمزامنة.
  • الموقع: لعرض أقرب العقارات (اختياري).
  • الكاميرا: للتوثيق وطلبات الصيانة ومسح QR.

7. الأطفال

التطبيق غير موجّه لمن هم دون 13 عامًا.

8. التحديثات

قد نحدّث هذه السياسة مع إصدارات التطبيق. استمرار الاستخدام يعني الاطلاع على النسخة الأحدث المنشورة هنا.
