# دليل استخدام لوحة تحكم الأدمن (Admin Dashboard)

لوحة تحكم مركزية لإدارة منصة "إيجاري" بالكامل، مع إحصائيات متقدمة وأدوات تحكم قوية.

## الميزات الرئيسية:

### 1. لوحة القيادة (Dashboard)
- **إحصائيات حية:** عدد المستخدمين، العقارات، الحجوزات، وإجمالي الإيرادات.
- **تنبيهات (Alerts):** إشعارات فورية بطلبات التحقق المعلقة والمشاكل المبلغ عنها.
- **النشاط الأخير (Recent Activity):** سجل بآخر العمليات التي تمت على المنصة.

### 2. الرسوم البيانية (Charts)
- **نمو المستخدمين:** رسم بياني يوضح زيادة عدد المستخدمين شهرياً.
- **الإيرادات:** مخطط خطي يوضح التدفقات المالية للمنصة.
- **توزيع العقارات:** مخطط دائري يوضح نسب أنواع العقارات (شقق، فيلات، محلات).

### 3. أدوات الإدارة (Management Tools)
- **إدارة المستخدمين:** عرض قائمة المستخدمين، تجميد حسابات، أو حذفها.
- **إدارة العقارات:** مراجعة العقارات الجديدة والموافقة عليها.
- **طلبات التحقق (Verification):** مراجعة وثائق الهوية والملكية للمستخدمين والمالك.
- **تصدير التقارير:** إمكانية تصدير البيانات (PDF/Excel) للتحليل (تجريبي).
