  • الأسعار بالجنيه المصري ما لم يُذكر خلاف ذلك. العقد النهائي هو المرجع لأي التزام مالي.

4. الإلغاء والاسترجاع

سياسات الإلغاء والاسترجاع تختلف حسب نوع الحجز أو الخدمة وتظهر داخل التطبيق قبل التأكيد. بعض الرسوم غير قابلة للاسترجاع وفق الشروط المعروضة.

5. الخصوصية

جمع واستخدام البيانات يخضع لـ سياسة الخصوصية.
